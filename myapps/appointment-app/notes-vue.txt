To install vue globally as node package
$npm install -g @vue/cli

To upgrade vue globally 
$npm install -g @vue/cli

To know the version of vue:
$vue --version 

To create new app:
$vue create hello-world

    Instead of 
        "vue create hello-world" use "vue ui" to run gui to get project/items get created 

To run vue app:
    $cd hello-world
        AND
        
    $npm run serve
        OR
    $vue serve 

To run on a port:
    $vue serve -- --port 3000
